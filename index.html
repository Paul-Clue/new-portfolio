Brain Tumor Classification

An interactive web application that uses deep learning to classify brain tumor MRI scans and provides expert-level analysis using OpenAI's GPT-4 Vision.

LLM Evaluation

This application evaluates the performance of LLMs on a variety of tasks, including code generation, natural language understanding, and question answering. It uses a variety of metrics to evaluate the performance of the LLMs. Type in a system for the llm to follow and a prompt to evaluate the llm on. You can then choose a model to evaluate. The application will then evaluate the llm on the prompt and provide the scores. A table with clickable cells is provided to view the results.

Codebase RAG

A sophisticated development tool that transforms how developers interact with codebases. By leveraging artificial intelligence and natural language processing, the application can analyze entire GitHub repositories, understand code structure, and provide intelligent insights. Built with Next.js and TypeScript for robust performance, the tool employs advanced language processing capabilities to parse code into Abstract Syntax Trees (ASTs), enabling deep understanding of code patterns and relationships.

Customer Churn Prediction

A little about me

Jamaica born - Brooklyn raised

I migrated from Jamaica to Brooklyn when I was six years old. Two years after migrating, my mother bought me a Nintendo. That beautiful Nintendo is what sparked my interest in coding. I very much loved to play the games, but I always wondered how they made them. In school I learned that people who worked as computer programmers were the creators of video games. It was that day in the fifth grade that I knew I wanted to be a computer programmer. However, the streets of Brooklyn had their own plans for me. Many years later, after learning some hard lessons and leaving the streets alone, I was able to settle down and focus on that nagging dream of becoming a computer programmer.

Did you hear about the rose that grew from a crack in the concrete? Proving nature’s laws wrong, it learned to walk without having feet. Funny, it seems to be keeping it’s dreams; it learned to breathe fresh air. Long live the rose that grew from concrete when no one else even cared.
